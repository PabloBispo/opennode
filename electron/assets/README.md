# Assets

Place application icons here:
- `icon.icns` — macOS app icon
- `icon.ico` — Windows app icon
- `icon.png` — Linux app icon (at least 512x512)

Icons are not included in the repository due to size. Generate them using the provided scripts or download them from the releases page.

## Tray Icons

Place tray icons here:
- tray-icon.png (16x16, 32x32) — default state
- tray-recording.png — recording active state (red dot)
- tray-icon@2x.png — retina macOS

For macOS menu bar, use Template images (monochrome, -Template suffix).
